# YouTube Video Manager (SQLite)

A simple Python CLI application to manage YouTube videos using SQLite database.

## Features
- Add new videos
- View all videos
- Update video details
- Delete videos
- Store data in SQLite database

## Requirements
- Python 3.x
- SQLite3 (comes with Python)

## Run the Project

```bash
python app.py
